A compact, reliable, and fully adjustable LED backup light that turns on instantly when the electricity goes out.
Key Features

Automatic switchover — LED lights up immediately during power failure (no delay, no relay)
Wide voltage support — Adjustable output from 3V to 12V using onboard multi-turn trimmer
Universal compatibility — Works with almost any LED, COB module, LED strip, or small panel
Safe Li-ion charging — Built-in TP4056 charger with proper current limiting
High power capable — Supports up to ~10–12W output
Added temperature monitoring — NTC thermistor for basic thermal protection
Clean & compact design — Single custom PCB (60×45mm)

Technical Specifications

Input: 5V USB-C
Battery: 3.7V Li-ion (18650 / 21700 recommended, protected cell preferred)
Output: 3V – 12V adjustable (via onboard 10k multi-turn trimmer)
Boost Converter: MT3608 (bare chip)
Automatic switching: AO3401 P-channel MOSFET
Charging: TP4056 with ~130mA safe charge current

Why Lumiere?
Most cheap emergency lights are fixed voltage and limited. Lumiere is designed to be truly flexible — you can connect virtually any LED light you have and fine-tune the exact voltage needed for optimal brightness and efficiency.
Perfect for frequent power cuts in Pakistan (and anywhere else).
